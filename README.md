# TÜİK Verileri ile Türkiye Alkol Tüketim Alışkanlıklarının Veri Bilimi Analizi

Bu proje, Veri Bilimi dönem sonu final ödevi kapsamında Türkiye'deki bireylerin alkol kullanma durumlarının cinsiyet ve yaş gruplarına göre zamansal (2019, 2022, 2025) değişimlerini incelemek ve modellemek amacıyla geliştirilmiştir. Geliştirme sürecinde "vibe coding" yaklaşımıyla yapay zeka asistanı birincil geliştirme ortağı olarak kullanılmıştır.

## 🛠 Kullanılan Kütüphaneler ve Teknolojiler
* Python 3.10+
* Jupyter Notebook
* Pandas & Numpy (Veri Maniplasyonu ve Temizliği)
* Matplotlib & Seaborn (Keşifsel Veri Analizi ve Görselleştirme)
* Scikit-Learn (Doğrusal Regresyon Modellemesi)

## 📊 Veri Kaynağı ve Lisans
* **Veri Seti:** TÜİK (Türkiye İstatistik Kurumu) - Türkiye Sağlık Araştırması (Bireylerin alkol kullanma durumunun cinsiyet ve yaş grubuna göre dağılımı)
* **Lisans:** TÜİK Açık Veri Portalı kullanım şartlarına tabidir.

## 🚀 Projeyi Çalıştırma Talimatları
1. Repository'i bilgisayarınıza indirin veya klonlayın.
2. Gerekli kütüphaneleri yükleyin: `pip install pandas numpy matplotlib seaborn scikit-learn xlrd`
3. `analiz.ipynb` dosyasını VS Code veya Jupyter Notebook üzerinden açarak yukarıdan aşağıya sırayla çalıştırın.
4. İlk kod bloğundaki alternatif veri seti yükleme satırına göre çalıştırınız.
